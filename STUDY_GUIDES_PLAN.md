## 3. ATI CMS Pediatrics Practice A Study Guide

### Web Research Summary (Quizlet + Studocu)

**High-Yield Topics (2025 Practice A):**

| Category | Key Concepts |
|----------|--------------|
| **Burns/Trauma** | Phases: emergent (fluids LR, airway), acute (infection, nutrition), rehab (ROM). Priority: pain control before dressing. Myoglobinuria in electrical burns. |
| **Respiratory** | Croup (racemic epi, steroids), asthma (silent chest = emergency), bronchiolitis (supportive), epiglottitis (do NOT examine throat). Tracheoesophageal fistula. |
| **GI** | Celiac (gluten-free), lactose intolerance, gastroenteritis (ORS preferred), appendicitis, pyloric stenosis, cleft palate (special nipples), biliary atresia. |
| **Infection** | Impetigo (contact), varicella, mononucleosis, precautions (air/droplet/contact). |
| **MSK/Neuro** | JIA (morning stiffness, ROM), meningitis (Kernig/Brudzinski), seizures, myelomeningocele (neurogenic bladder). |
| **Heme/Onc** | Leukemia, sickle cell crisis, hemophilia, acute hemolytic transfusion reaction, chemo extravasation. |
| **Endocrine** | Type 1 diabetes (sick days, hypo/hyper), congenital hypothyroidism, FTT. |
| **Dev/Behavior** | Autism (short visits), age-specific teaching (toddler→adolescent). |
| **Meds/Skills** | Weight-based dosing, eardrops (tragus), eyedrops (conjunctival sac), epi teaching. |

**Priority Nursing Frameworks:**
- ABCs, Maslow, least restrictive
- Age-appropriate care (Erikson/Piaget)
- Family-centered care
- Safety (infection, injury prevention)
